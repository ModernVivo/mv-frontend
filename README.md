# mv-frontend

## Required
- Nodejs
- yarn or npm

## Install & run on local

- cd to project
> cd mv-fontend

- Copy env.example to .env
> cp .env.example .env

- Install node_module

With yarn

> yarn install

With npm

> npm install

- Start dev local

With yarn

> yarn dev

With npm

> npm run dev
